ヘルプステーションわかたけ

ヘルプステーションわかたけ

お買い物も、家事も、
安心安定の生活を

ヘルプステーションわかたけは、日常生活を支援するにあたり、数々の支援種別に対応しています。

居宅介護

ご自宅での暮らしを支えるために、生活全般を包括的にサポートします。
利用者の健康管理や安全な生活環境の提供をはじめ、個別のニーズに対応するサービスを通じて、 地域社会の中で安心して暮らせる環境づくりをお手伝いします。

サービス内容

身体介護 / 家事援助 / 通院介助

重度訪問介護

重度訪問支援では、重度の障がいをお持ちの方が自宅で安心して生活できるよう、
日常のケアから生活環境の整備まで幅広い支援を提供しています。

サービス内容

日常生活全般の介護 / 見守りや相談支援

同行支援

同行支援は、移動が不安な方に寄り添い、外出の機会をサポートします。
生活をより豊かにするために、通院や買い物から余暇活動まで幅広い場面でお手伝いします。

サービス内容

通院、買い物、行政手続き等の外出支援 / 趣味や余暇活動、社会参加への外出の同行 / 安全に移動できるよう寄り添う支援

日中移動支援

日常生活を豊かにするための外出を支援するのが、日中移動支援です。
移動が困難な方でも安心して利用できる環境を整えています。

サービス内容

通院や買い物など日常生活に必要な移動支援

行動支援

行動支援では、利用者が安心して日常生活を送れるよう行動のサポートを行います。
安全な環境の整備や個別対応を通じて、利用者の自立を支援します。

サービス内容

危険行動や不適切な行動を防ぐためのサポート / 行動を見守り、安心して生活できる環境を整備

移動支援

移動の不安を軽減し、外出の機会を広げる移動支援。
利用者の生活をより充実させるお手伝いをします。

サービス内容

障がい特性に配慮した安全な移動手段の提供 / 買い物や趣味活動、地域イベント参加の支援 / 通院や行政手続きのための移動支援

対応可能エリア

大津市 / 草津市 / 栗東市 / 守山市 / 野洲市

場合によっては対象エリア外にも対応可能なことがございますので、まずはご相談ください。
